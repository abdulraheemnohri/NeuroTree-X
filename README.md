# 🌌 NeuroTree X
### "A Living AI Brain That Learns, Connects, and Evolves."

NeuroTree X is a next-generation Autonomous AI Knowledge Graph Platform. It converts raw information into an evolving visual knowledge tree with infinite nodes, AI memory, and autonomous research capabilities.

## 🚀 Key Features
- **Visual Knowledge Tree**: Interactive 3D visualization using Three.js and React Three Fiber.
- **Autonomous AI Research**: Multi-agent system (Scientist, Explorer, Critic) collaborating in a dedicated Lab.
- **Self-Learning Evolution**: Automatic pattern discovery and knowledge reinforcement via the Learning Engine.
- **Digital Twins & Simulation**: Validate research in interactive 3D/VR worlds with live entity mirroring.
- **Personal AGI Assistant**: Proactive terminal-style task management and contextual support.
- **Code Synthesis**: Autonomous software creation (Python/Rust) from graph specifications.
- **Distributed Network**: P2P knowledge sharing via IPFS, libp2p, and CRDT-based merging.
- **Multi-modal Indexing**: Neural indexing for visual media and complex documentation.
- **Local AI Control**: Management of local LLMs (Gemma, Llama, Mistral) with GPU optimization.

## 🏗️ Architecture
The system is built as a distributed microservices ecosystem (20+ core modules):
- **Backend**: FastAPI services orchestrating intelligence, simulation, and P2P sync.
- **Frontend**: Next.js + Tailwind + Three.js for the premium glassmorphism UI.
- **Databases**: Neo4j (Graph), Qdrant (Vector), Redis (Cache), IPFS (P2P).
- **Platforms**: Web, Desktop (Tauri), and Mobile (React Native).

## 🛠️ Getting Started
Run the bootstrap script to initialize the environment:
\`bash
./scripts/bootstrap.sh
\`

Then use Docker Compose to start the full system:
\`bash
docker-compose up -d
\`
